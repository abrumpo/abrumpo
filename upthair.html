upThair

upThair by abrumpo is an aggregator of cloud services. Version 1.2 is focused on file management within and across multiple cloud services. Within a service, it allows users to view iOS supported file types, view and save files offline, rename, copy, move, delete, and upload files.* upThair also allows users to search for and copy files across cloud services. Additionally, users can view, start, stop, and reboot their Amazon Web Services (AWS) EC2 servers.

Supported cloud services include:
Amazon Web Services (AWS) S3 & EC2
Dropbox
Google Drive
box
OneDrive

*upThair does not support file editing capability
